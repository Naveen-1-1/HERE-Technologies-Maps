# 1. HERE-specific mapmaking context

1.1. [HERE UniMap / AI mapmaking](https://www.here.com/learn/blog/AI-mapmaking)

- AI to make quick calculations from billions of possibilities to solve difficult problems, including ETA calculation. 
- Autonomous driving: smaller AI solutions solving the bigger problem, object detection & image recognition unlikely to be perturbed by Gen AI

1.2. [HERE map-change validation](https://www.here.com/learn/blog/how-to-properly-validate-a-map-change)

- Maps don't just exist they consistently evolve hence data collection required
- Data collection done via companies, users, data providers, governments, etc.
- Updates from these sources need to be cross checked/verified with other sources
- Then the optimized system along with respective company (BMW, Audi, Microsoft, etc) specific data is sent back for building various custom map tools/technologies
